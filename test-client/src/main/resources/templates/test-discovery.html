说明

OIDC Discovery 端点: GET /.well-known/openid-configuration

此端点返回 OpenID Connect Provider 的元数据，包括：

  • issuer - 发行者 URL
  • authorization_endpoint - 授权端点
  • token_endpoint - Token 端点
  • userinfo_endpoint - 用户信息端点
  • jwks_uri - JWKS 公钥端点
  • 支持的 grant types、response types、scopes 等
