说明

Client Credentials 流程适用于机器对机器（M2M）的认证场景，不涉及用户。

端点: POST /oauth2/token

请求参数:

  • grant_type=client_credentials
  • scope - 请求的权限范围
  • 通过 Basic Auth 发送 Client ID 和 Secret

默认客户端: demo-client / demo-secret
